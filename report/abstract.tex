\begin{abstract}

\texttt{GuitarDSP} is an application written in Java that allows to
modify a sound file in real time, applying on it some effects.\\
The ultimate purpose is to simulate a multi-effect pedal for guitar but
due for time this version of the project allows user to modify
an audio source taken by a .wav file stored on the pc.
\end{abstract}
